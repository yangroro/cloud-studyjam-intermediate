## 진행방식

- 온라인(가능하면 웹캠 키고 진행)
- 스프린트 목표 설정 후 각자 정해진 분량을 진행하고 모여서 질문과 노트 작성하기
  - 질문이나 노트는 GitHub Issue로 작성
  - 참여자는 스프린트마다 최소 1개의 질문이나 노트 작성하기
- 매 스터디마다 1명씩 돌아가면서 스터디 내용 정리하는 서기 역할하기
  - 매 스터디마다 카톡방에 모인 참고자료 링크들 취합해서 서기가 GitHub에 업로드하기
